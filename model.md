Model
Backend pada aplikasi ini menggunakan bahasa pemrograman python dan database MongoDB.
Terdapat empat kelas model yaitu Admin, staff, tamu, dan barang.
 • Admin memiliki attribut id, nama, nomor telepon, dan email.
 • Staff memiliki attribut id, nama, nomor telepon, alamat, posisi, dan email.
 • Tamu memiliki attribut id, nama, nomor telepon, alamat, dan email.
 • Barang memiliki attribut id, nama, jenis, stok, satuan, dan harga.
Terdapat operasi untuk melakukan pengelolaan database barang.
 • Operasi untuk membuat database barang (barang.db)
 • Operasi penambahan barang dengan memasukkan id_barang, nama_barang, jenis_barang, stok,    satuan, dan harga_barang
 • Operasi untuk melihat seluruh daftar barang pada barang.db
 • Operasi untuk menghapus barang berdasarkan id_barang
 • Operasi untuk update barang berdasarkan id _barang
